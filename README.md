# Django Server for VulcanX

## Installation
  * Activate VirtualEnv
    
    For macOS\Linux:
    ```
    $ source venv/bin/activate
    ```

    For the other thing:
    ```
    $ venv\Scripts\activate
    ```

  * Install required packages
    ```
    $ pip install -r requirements.txt
    ```

> Before running always activate VirtualEnv
